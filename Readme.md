# Инструкция по работе с git

## Создание репозитория
Для того что бы добвыить версионность к созданой папке, и создать в ней локальный репозиторий, для этого необходимо открыть окно терминала в этой папке и написать команду "git init". Тогда ваша папка станет репозиторием, и в ней появится скрытая папка .git
